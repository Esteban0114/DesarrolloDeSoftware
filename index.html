¿Qué es el Desarrollo de Software?

Es el conjunto de actividades informáticas dedicadas al proceso de creación, diseño, despliegue y compatibilidad de software. El software es el conjunto de instrucciones o programas que le dicen a una computadora qué hacer.

Software de Sistema

Proporciona funciones básicas (Ej: Windows, Linux).

Software de Aplicación

Ayuda a realizar tareas específicas (Ej: WhatsApp, Excel).

El Ciclo de Vida (SDLC)

Las fases esenciales para crear software de calidad.

01

Análisis de Requisitos

Definir qué necesita el cliente y qué hará el software. Es la base de todo el proyecto.

02

Diseño (UI/UX y Arq)

Se definen la arquitectura técnica y la interfaz gráfica. Cómo se verá y cómo funcionará por dentro.

03

Desarrollo (Coding)

Los programadores escriben el código utilizando lenguajes como Python, Java, C++ o JavaScript.

04

Pruebas (Testing)

Se buscan errores (bugs) y se verifica que el software cumpla con los requisitos iniciales.

05

Despliegue y Mantenimiento

Se entrega al usuario final y se realizan actualizaciones constantes para mejorarlo.
